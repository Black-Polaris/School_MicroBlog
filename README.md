# School_MicroBlog
## 基于SpringBoot+Vue+Mybatis+Redis的前后端分离的校园微博系统 *后端部分*
[前端代码跳转链接](https://github.com/Black-Polaris/School_MicroBlog_Client)  [https://github.com/Black-Polaris/School_MicroBlog_Client](https://github.com/Black-Polaris/School_MicroBlog_Client)

## 系统的主要实现功能
  * 通过Redis实现热门微博和微博热搜
  * 使用ElementUI进行前端页面ui设计
  * 使用WebSocket实现私聊和群聊功能
  * 应用JWT作为用户的身份认证
  * 使用Shiro实现用户登录认证
